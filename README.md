# Product Listing Page

## Run Locally

Clone the project

```bash
  git clone https://github.com/habib33-3/product-listing-page
```

Go to the project directory

```bash
  cd product-listing-page
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
